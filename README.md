I.Giới thiệu
Project StudentManager là ứng dụng quản lý sinh viên đơn giản, được xây dựng bằng C# WinForms.
Ứng dụng cho phép thực hiện các chức năng cơ bản như:
Thêm sinh viên
Cập nhật thông tin
Xóa sinh viên
Tìm kiếm sinh viên
Kiểm tra dữ liệu đầu vào (validation)
Dự án được sử dụng để thực hành Git và GitHub theo quy trình làm việc chuẩn của lập trình viên.


II.Cách chạy ứng dụng

b1:clone về máy qua git clone https://github.com/wfign16zy3elx78p-ship-it/HuynhHuyCuong_21484_StudentManager

b2:Mở file .sln bằng Visual Studio

b3:Chọn Start để chạy chương trình
